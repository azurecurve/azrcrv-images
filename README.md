# [Images](https://development.azurecurve.co.uk/classicpress-plugins/images/)
![Plugin Banner](/assets/pluginimages/banner-1544x500.png)

# Description

Shortcode to display image in post; add width, alt, caption or error attributes.

Example shortcode:
```
[image image="http://www.example.com/hello.png" width="550px" alt="Hello World!"]
```

This plugin is multisite compatible.

# Installation Instructions

 * Download the latest release of the plugin from [GitHub](https://github.com/azurecurve/azrcrv-images/releases/latest/).
 * Upload the entire zip file using the Plugins upload function in your ClassicPress admin panel.
 * Activate the plugin.
 * Configure relevant settings via the configuration page in the admin control panel (azurecurve menu).
